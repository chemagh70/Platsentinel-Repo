🚀 Guía Visual: Entorno Común con Docker y Docker Compose

🎯 ¿Qué incluye este entorno?

  • Kali Linux: contenedor de terminal para prácticas de ciberseguridad
  • Cockpit: interfaz gráfica web para gestionar el servidor
  • OWASP Juice Shop: aplicación vulnerable para pruebas
  • PostgreSQL: base de datos para test y conexión desde scripts

📦 Pasos para desplegar el entorno

  1. Instala Docker y Docker Compose en tu equipo
  2. Coloca el archivo docker-compose.yml en una carpeta vacía
  3. Abre terminal en esa carpeta y ejecuta:
docker-compose up -d

🌐 Accesos y servicios

  • Cockpit (interfaz web): http://localhost:9090
  • Juice Shop: http://localhost:3000
  • PostgreSQL: puerto 5432, usuario: admin, contraseña: admin123
  • Kali Linux: contenedor accesible con: 
    docker exec -it kali_dev bash

👥 ¿Cómo conectarse desde diferentes sitios?

Si estáis en redes distintas, se recomienda una de estas opciones seguras:

  • ZeroTier: Red privada virtual gratuita. Todos los usuarios se conectan como si estuvieran en LAN.
  • Tailscale: VPN basada en WireGuard con control de acceso por usuarios.
  • Servidor Cloud (opcional): Subir este entorno a una máquina en la nube (ej. Oracle Cloud, AWS Free Tier).

Para acceso individual, se pueden crear usuarios dentro de Cockpit o en el propio sistema Ubuntu base.
